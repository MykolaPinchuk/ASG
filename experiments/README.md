# Experiments

This directory holds committed, human-readable experiment packs.

`POLICY.json` defines default seed profile and baseline control-rerun cadence.

Each ablation should have one folder:
- `EXP_###_<slug>/experiment.json`
- `EXP_###_<slug>/conditions/control.json`
- `EXP_###_<slug>/conditions/variant.json`
- `EXP_###_<slug>/artifacts/<condition>/prompts/*.txt`
- `EXP_###_<slug>/artifacts/<condition>/rules_snapshot.md`
- `EXP_###_<slug>/results/state_diff.md`
- `EXP_###_<slug>/results/latest.md`
- `EXP_###_<slug>/results/replay_index.csv`
- `EXP_###_<slug>/results/interpretation.md` (short decision note; generated by `npm run exp:report`)

`results/latest.md` links to run logs under `runs/experiment_logs/...`.

Top-level indexes:
- `experiments/INDEX.md`
- `experiments/INDEX.csv`
